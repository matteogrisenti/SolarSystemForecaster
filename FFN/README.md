# FFN for Solar Power Prediction
This project implements an estimator for solar power production based on a trained Feedforward Neural Network (FFN). The estimator predicts solar system power output given input features (e.g., weather conditions, time of day, etc.) using a model trained in Google Colab

## Folder Structure
```
dataset/
│
├── FFN.ipynb                          # Jupyter notebook used to train the FFN model in Colab (v6e-1 TPU runtime)
├── FFN_estimator.py                   # Implements the SolarPowerEstimator class for predictions
│
├── train/                             # Training results, including the trained model
│
├── test/                              
│   ├── estimator_test.py              # Unit tests for the estimator
│   ├── test_predictions.csv           # Predictions on the X_test dataset
│   ├── test_evaluation.py             # Evaluation script comparing predictions to ground truth (y_test)
│   ├── fnn_prediction_evaluation.txt  # Evaluation report generated by test_evaluation.py
│
├── README.md                          # Project overview and instructions
```

## Estimator Usage
1. Add the estimator to your Python script:
```bash 
import sys, os
sys.path.append(os.path.abspath(os.path.join(os.path.dirname(__file__), '..')))
from FFN_estimator import SolarPowerEstimator
```
2. Create an instance and use the estimator:
```bash 
# Initialize the estimator (make sure the model path is correct)
estimator = SolarPowerEstimator(model_path="train/your_trained_model.pth")

# Predict power output for a given dataset
predictions = estimator.predict(X_input)
```

## Training the Model
1. Open FFN.ipynb in Google Colab (TPU runtime v6e-1 recommended).
2. Upload the dataset csv files in dataset/train_test_split
3. Follow the notebook to preprocess data, train the FFN, and save the trained model.
